# Integration Plan — MinhaFila Saúde

## Objetivo
Organizar a integração entre app Flutter, backend REST e painel administrativo.

---

## Etapa 1 — Backend base
- subir API REST;
- criar banco homologado;
- implementar auth mock;
- implementar queue requests;
- implementar history;
- implementar notifications.

## Etapa 2 — Validação
- criar validation_requests;
- criar attachments;
- alterar status no backend;
- devolver atualização ao app.

## Etapa 3 — Painel admin
- login admin;
- lista de pendências;
- detalhe com anexo;
- aprovar/rejeitar.

## Etapa 4 — Integração completa no app
- remover mocks principais;
- consumir API real;
- tratar loading, erro e estados vazios.

## Etapa 5 — Documentação e demonstração
- atualizar docs;
- preparar base homologada;
- demonstrar fluxo ponta a ponta.
